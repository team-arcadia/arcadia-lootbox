# Arcadia LootBox

![License](https://img.shields.io/badge/license-All%20Rights%20Reserved-red) ![Version](https://img.shields.io/badge/version-1.1.0-blue)

**[EN]** Advanced lootbox system with custom visuals and configuration for the Arcadia server.
**[FR]** Système de lootbox avancé avec visuels personnalisés et configuration pour le serveur Arcadia.

---

## 🇺🇸 English (US)

### Features
- **Custom Lootboxes**: Create unlimited lootboxes via JSON configuration.
- **Physical Keys**: Define any item (vanilla or modded) as a key to open a lootbox.
- **Visual Effects**: Customizable particles and sounds upon opening.
- **Dynamic Content**: Set loot tables with probabilities, min/max counts.
- **Bilingual**: Fully translated in English and French.

### Commands
- `/arcadialoot give <target> <lootbox_id>` *(Admin)*: Gives a lootbox to a player.
- `/arcadialoot reload` *(Admin)*: Reloads configuration from disk.

---

## 🇫🇷 Français (FR)

### Fonctionnalités
- **Lootboxes Personnalisées** : Créez des lootboxes illimitées via configuration JSON.
- **Clés Physiques** : Définissez n'importe quel item (vanilla ou moddé) comme clé pour ouvrir une lootbox.
- **Effets Visuels** : Particules et sons personnalisables à l'ouverture.
- **Contenu Dynamique** : Définissez des tables de butin avec probabilités, quantités min/max.
- **Bilingue** : Entièrement traduit en Français et Anglais.

### Commandes
- `/arcadialoot give <joueur> <lootbox_id>` *(Admin)* : Donne une lootbox à un joueur.
- `/arcadialoot reload` *(Admin)* : Recharge la configuration depuis le disque.
